# 캠핑카/카라반 옵션표 생성기

웹 기반 캠핑카/카라반 옵션표 생성 도구입니다. 입력 폼에서 데이터를 입력하면 정사각형 PNG 이미지로 다운로드할 수 있습니다.

## 기능

### 캠핑카 탭
- **차량 정보**: 베이스 차량, 제조사, 모델명, 차종, 연식, 최초등록일, 주행거리, 차고지 증명, 필요 면허
- **제원**: 길이, 너비, 높이, 배기량, 연료, 변속기, 연비, 승차정원, 현금 영수증
- **전기**: 배터리(인산철/딥싸이클 + 용량), 태양광, 인버터
- **옵션**: 외관, 내장, 편의

### 카라반 탭
- **차량 정보**: 제조사, 모델명, 차종, 연식, 최초등록일, 차고지 증명, 취침인원, 현금 영수증
- **제원**: 외부/내부 길이, 외부/내부 너비, 외부/내부 높이, 공차 중량, 최대 허용 중량
- **전기**: 배터리, 태양광, 인버터
- **옵션**: 외관, 내장, 편의

## 자동 포맷팅

### 연식 파싱
- `2021` 입력 → 항목명: "연식", 값: "2021 년"
- `2021.3` 입력 → 항목명: "제작연월", 값: "2021 년 3 월"

### 단위 자동 추가
- 주행거리: `35000` → `35,000 km`
- 길이/너비/높이: `7315` → `7,315 mm`
- 배기량: `2497` → `2,497 cc`
- 중량: `1450` → `1,450 kg`
- 인원: `4` → `4 인`
- 배터리: 타입 선택 + `300` → `인산철 300 Ah`
- 태양광: `180` → `태양광 180 W`
- 인버터: `3` → `인버터 3 Kw`
- 연비: `8.5` → `등록증상 8.5 km/L`

### 옵션 포맷팅
- 스페이스 2번으로 구분하면 자동으로 `• ` 변환
- 예: `청수 주입구  한전 인입구` → `• 청수 주입구 • 한전 인입구`

## 기타 기능

- **로컬스토리지 자동 저장**: 새로고침해도 입력값 유지
- **초기화 버튼**: 확인 다이얼로그 포함
- **모달 미리보기**: 결과를 전체 화면 모달로 표시
- **PNG 다운로드**: 정사각형 이미지로 다운로드 (흰색 배경 포함)
- **파일명**: `{모델명}_옵션표.png`

## 기술 스택

- 순수 HTML/CSS/JavaScript (프레임워크 없음)
- html2canvas 라이브러리 (CDN)
- Pretendard 폰트
- 로컬스토리지 API

## 사용법

1. HTML 파일을 브라우저에서 열기
2. 캠핑카 또는 카라반 탭 선택
3. 필요한 정보 입력
4. "결과 미리보기" 클릭
5. "PNG 다운로드" 클릭

## 디자인

- 네이비 블루 (#1e3a5f) 헤더
